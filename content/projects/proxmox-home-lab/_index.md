A full security lab built on Proxmox VE — documented from scratch, including the mistakes.

The lab covers network segmentation with OPNsense, a Wazuh SIEM for log collection and alerting, a Windows Active Directory domain, and hardened Linux servers. The goal is a realistic environment for practising both defensive and offensive security.

This project is documented in two parallel tracks — one manual, one AI-assisted.
